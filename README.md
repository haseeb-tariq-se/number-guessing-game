# Number Guessing Game (Java)

A simple Java console-based game where the player tries to guess a randomly generated number within a limited number of attempts.

## Overview
This project was built to practice Java programming fundamentals, logical thinking, problem solving, and modular programming concepts.

The application generates a random number between 1 and 100, and the player must guess the correct number. After every guess, the program provides feedback indicating whether the guess is too high or too low. The game also includes a scoring system, multiple rounds support, and a final performance summary.

## Features
• Random number generation  
• Interactive console-based gameplay  
• Higher/lower feedback system  
• Score calculation system  
• Multiple rounds support  
• Attempt limit for each round  
• Final score and average score display  
• Beginner-friendly and modular code structure

## How to Run the Project
1. Download or clone the repository  
2. Open the project in IntelliJ IDEA or any Java IDE  
3. Compile the Java file  
4. Run `Number Guessing Game` file
5. Follow the console instructions to play the game  

## Game Rules
• The program generates a random number between 1 and 100  
• The player has a maximum of 7 attempts to guess the number  
• Feedback is provided after every attempt  
• Correct guesses award points based on remaining attempts  
• If all attempts are used, the correct number is revealed
